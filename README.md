## Descrição

Microserviço de frontend para gerenciar o login de usuários e que compartilha os dados do usuário logado para os 2 outros microfrontends.

## Setup do projeto

```bash
$ npm install
```

## Compilar e rodar o projeto

```bash
# development
$ npm run dev
```
